Quickstart

In this page you can find some example of use of TimeSource interface and its implementations. Samples are provided in form of unit test snippets.

ConstantTimeSource

Initializing ConstantTimeSource providing a long value:

	private static final long TIME_IN_MILLIS = 12345L;

		final TimeSource source = new ConstantTimeSource(TIME_IN_MILLIS);

Getting currentTimeMillis from ConstantTimeSource:

		assertEquals(TIME_IN_MILLIS, source.currentTimeMillis());

Getting currentTime from ConstantTimeSource:

		assertEquals(TIME_IN_MILLIS, source.currentTimeMillis());

Initializing ConstantTimeSource providing a java.util.Date instance:

Finally, you can initialize new ConstantTimeSource instance providing a java.util.Date instance:

		final Date aDate = new Date();
		final TimeSource source = new ConstantTimeSource(aDate);

SequentialTimeSource

Initializing SequentialTimeSource providing long values:

	private static final long T0 = 0;
	private static final long T1 = 10;
	private static final long T2 = 100;

		TimeSource source = new SequentialTimeSource(T0, T1, T2);

Initializing SequentialTimeSource providing java.util.Date instances:

	private static final Date D0 = new Date(T0);
	private static final Date D1 = new Date(T1);
	private static final Date D2 = new Date(T2);

		TimeSource source = new SequentialTimeSource(D0, D1, D2);
